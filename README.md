# BIOCORE Product Website

専門学校のWeb制作課題として制作した、スマートデバイスブランド「BIOCORE」の架空プロダクトサイトです。支給されたデザイン案をもとに、PC・タブレット・スマートフォンで閲覧できるレスポンシブな1ページサイトとして実装しています。

## 制作概要

- 制作目的：企業向けポートフォリオに掲載できるWebサイトの制作
- ページ構成：ブランド紹介、製品紹介、実績、ユーザーレビュー、サポート、フッター
- 対応画面：PC、タブレット、スマートフォン
- 使用技術：HTML5、CSS3

## 工夫した点

- デザイン案の余白、配色、文字サイズ、カード比率をCSSで再現
- 固定幅のレイアウトを見直し、可変幅コンテナと複数のブレークポイントを採用
- 見出し階層、ナビゲーション、リスト、引用などを意味に合ったHTML要素で整理
- キーボード操作用のスキップリンク、フォーカス表示、画像の代替テキストを設定
- ページ内リンクを各セクションへ接続し、存在しないページへのリンクを回避
- `prefers-reduced-motion` に対応し、動きを抑えたい閲覧環境へ配慮

## 素材について

この作品では、課題制作のためにAI生成画像および無料配布のUI素材・アイコンを使用しています。画像とアイコンは学習・ポートフォリオ用途を想定したものであり、実在する製品・企業・サービスとは関係ありません。

素材ごとの配布元やライセンス情報が元データに含まれていないため、商用利用や再配布を行う場合は、使用条件と権利関係をあらためて確認する必要があります。ロゴおよび「BIOCORE」の名称も本課題用の架空ブランドです。

## 閲覧方法

`index.html` をブラウザで開くと閲覧できます。ビルド作業や外部ライブラリのインストールは不要です。
