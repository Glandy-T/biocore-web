# BIOCORE Webサイト

専門学校の授業課題として制作した、架空の未来ロボット販売サイトです。  
「BIOCORE」という架空ブランドを想定し、人々の日常にあるさまざまな困りごとや需要を、ロボットやスマートデバイスによって支える世界観でデザインしました。

## 制作内容

未来的で清潔感のある印象になるように、白と青を中心にした配色を使い、製品紹介サイトとして見やすい構成を意識しました。  
トップページ、製品紹介、特徴紹介、レビュー、サポートなどの内容を1ページのWebサイトとしてまとめています。

## 使用技術

- HTML
- CSS

## 制作で意識したこと

デザイン案をもとに、余白や文字の大きさ、画像の見せ方を調整しながら、サイト全体の雰囲気が統一されるように制作しました。  
また、見る人が製品の特徴を理解しやすいように、情報の順番やセクションの切り替わりを意識しました。

## この制作で学んだこと

Webサイトを制作する中で、見た目を整えるだけではなく、情報をどの順番で伝えるか、どのように見せると分かりやすいかを考えることの重要性を学びました。  
また、素材やデザインの雰囲気を活かしながら、架空の商品やブランドの世界観をWebページとして形にする経験ができました。

## 素材について

画像および無料配布のUI素材・アイコンを一部使用しています。
